# Old Hiro Website Backup

This folder preserves the previous long-form Hiro website design from the live `hiroagency.com` page before the Mainframe-style hero replacement.

- `index-old-design.html`: previous site HTML, including its inline JavaScript.
- `assets/styles.css`: previous reachable custom CSS from `hiroagency.com/assets/styles.css`.
- `assets/images/` and `assets/favicon_io/`: local assets referenced by the old HTML.

The old live HTML referenced `/index.css` and `/index.tsx`, but both return 404 on the deployed site, so they are not included as backup files.
